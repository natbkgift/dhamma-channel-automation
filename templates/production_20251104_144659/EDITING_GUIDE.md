## ðŸ› ï¸ DaVinci Resolve Workflow

### 1. Setup Project
- Create new project: `Dhamma_Video_001`
- Settings: 1920x1080, 30fps, Rec.709

### 2. Import Media
- Import voiceover audio: `audio/voiceover.mp3`
- Import all B-roll videos from `broll/` folder

### 3. Edit Timeline
- Place voiceover on Audio Track 1
- Use this guide to place B-roll on Video Track 1
- Add text overlays using Fusion > Text+

### 4. Color Grading
- Apply consistent color grade across all clips
- Warm tones for meditation scenes
- Soft contrast

### 5. Export
- Format: MP4 (H.264)
- Resolution: 1080p or 4K
- Bitrate: 10-15 Mbps (1080p) or 35-45 Mbps (4K)
- Audio: AAC 192 kbps
